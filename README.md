# valantic-cordova-plugin-zebra-printer

Cordova plugin that wraps the **Zebra Link-OS SDK** to allow
WiFi/Bluetooth printing from Android & iOS apps.  
Built for Valantic / OutSystems mobile apps.

## Install

```bash
cordova plugin add https://github.com/valantic/valantic-cordova-plugin-zebra-printer.git
# or from npm when published
# cordova plugin add valantic-cordova-plugin-zebra-printer
